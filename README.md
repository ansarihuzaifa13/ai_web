# 🚀 AI Dashboard Web App (Flutter)

A modern, high-performance **AI-powered dashboard web application** built using **Flutter Web**, focused on delivering a **premium UI (Dribbble-level design)** with smooth interactions and scalable architecture.

---

## 🌐 Live Demo

🔗 https://ansarihuzaifa13.github.io/ai_web/

---

## 📌 Overview

This project showcases a visually rich and responsive dashboard interface designed for AI-driven platforms.
It emphasizes **clean architecture**, **reusable components**, and **high-end UI/UX design principles**.

---

## ✨ Features

* 🎨 **Premium UI Design**

  * Modern layout inspired by Dribbble designs
  * Gradient themes, glassmorphism & clean typography

* 📊 **Dashboard Components**

  * Analytics cards
  * Data visualization sections
  * Structured information hierarchy

* ⚡ **High Performance**

  * Optimized Flutter Web build
  * Tree-shaken assets for faster load time

* 📱 **Responsive Layout**

  * Works across desktop & tablet screens

* 🧩 **Modular Architecture**

  * Feature-based folder structure
  * Scalable and maintainable code

---

## 🏗️ Tech Stack

* **Framework:** Flutter (Web)

* **State Management:** (Can be extended to Bloc / Provider)

## 🚀 Getting Started

### 1️⃣ Clone the repository

```bash
git clone https://github.com/ansarihuzaifa13/ai_web.git
cd ai_web
```

### 2️⃣ Install dependencies

```bash
flutter pub get
```

### 3️⃣ Run locally

```bash
flutter run -d chrome
```

---

## 🌍 Build for Web

```bash
flutter build web --base-href "/ai_web/"
```

---

## 🚀 Deployment (GitHub Pages)

This project is deployed using **gh-pages branch**.

```bash
dart pub global run peanut --release
git push origin gh-pages
```
